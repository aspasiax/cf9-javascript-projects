# 01 - Counter App

A clean and functional counter application developed as part of my JavaScript studies. This project focuses on DOM manipulation and implementing a structured code architecture.

## Key Concepts
In this project, I applied the following concepts:
* **MVC Pattern**: Separated the logic into Model (state), View (UI updates), and Controller (event handling).
* **DOM Events**: Utilized `DOMContentLoaded` and event listeners for interactive buttons.
* **State Management**: Handling a simple numeric state and reflecting changes dynamically.

## Features
* **Increase/Decrease**: Adjust the counter value.
* **Reset**: Return to the default value (0).
* **Dynamic Styling**: The counter color changes automatically (Green for positive, Red for negative, Black for zero).

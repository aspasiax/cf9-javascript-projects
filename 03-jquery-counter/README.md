# 03 - jQuery Counter

A variation of the Counter application, this time implemented using the **jQuery library** to handle DOM manipulation and event listeners.

## Key Concepts
* **jQuery Integration**: Leveraging the `$` syntax for efficient element selection and event handling.
* **MVC Pattern**: Maintaining a clean separation between state logic (Model), UI logic (View), and user input (Controller).
* **Class Toggling**: Using jQuery's `.toggleClass()` method for dynamic CSS state management.
* **Legacy Support Skills**: Demonstrating familiarity with widely-used JavaScript libraries.

## Features
* Standard Increase, Decrease, and Reset functionality.
* Smooth UI transitions and color-coded states.
* Cross-browser event handling simplified through jQuery.

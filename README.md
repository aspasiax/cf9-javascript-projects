# CF9 JavaScript Projects

This repository contains a collection of projects developed during the **Coding Factory 9** professional certification program. Each project demonstrates core JavaScript principles and modern web development practices.

## Projects Included

* **01-counter-app**: A simple counter focusing on the MVC pattern and basic DOM manipulation.
* **02-notes-app**: A task management application utilizing immutable state and Tailwind CSS.
* **03-jquery-counter**: A variation of the counter app, demonstrating usage of the jQuery library.
* **04-movies-app**: A dynamic movie search tool using the OMDB API and Axios for asynchronous data fetching.

## Technical Stack
* **Language:** JavaScript (ES6+)
* **Libraries:** jQuery, Axios
* **Styling:** Tailwind CSS, Bootstrap 5
* **Documentation:** JSDoc (English)

## Key Learning Outcomes
* Implementation of the **Model-View-Controller (MVC)** architecture.
* Handling **Asynchronous** operations and API integrations.
* Managing **Immutable State** and browser events.
* Professional documentation and clean code practices.

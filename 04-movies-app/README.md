# 04 - Movie Search App

A dynamic movie discovery tool that fetches real-time data from the **OMDB API**. This project highlights advanced asynchronous JavaScript techniques and UI state management.

## Key Concepts
* **Asynchronous API Calls**: Utilizing **Axios** for robust HTTP requests.
* **Debouncing Logic**: Optimized search performance by limiting API calls while the user is typing.
* **UI State Handling**: Managing Loading (spinner), Error, and "Not Found" states dynamically.
* **jQuery Animations**: Using `slideToggle` for a smooth disclosure of extended movie details.
* **Responsive Design**: Styled with **Tailwind CSS** for a modern, mobile-friendly look.

## Features
* Live search with auto-fetch after typing.
* Detailed movie statistics (IMDb rating, Box Office, Cast).
* Direct link to IMDb profiles.
* Visual loading indicators and error feedback.

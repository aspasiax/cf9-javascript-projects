# 02 - Notes App

A task management application built with **Vanilla JavaScript** and styled with **Tailwind CSS**. This project demonstrates state management and dynamic UI rendering.

## Key Concepts
* **Immutable State**: Handling notes using spread operators and array methods (`map`, `filter`).
* **Dynamic Rendering**: Generating HTML elements on the fly based on the application state.
* **Internationalization**: Utilizing the `Intl` API for localized date and time formatting.

## Features
* **CRUD Operations**: Add and delete notes.
* **Toggle Status**: Mark notes as completed with a strike-through effect.
* **Real-time Clock**: A live-updating header showing the current date and time.
